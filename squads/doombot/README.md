```bash

*rewrite copy-fraco

*war-room contatos dor

# Quality Gate
*quality-gate entregavel
```

## Workflows

1. **Full Campaign Pipeline** — `diagnose → plan → offer → copy → tension → QG → launch → measure → LTV`
2. **Copy Review Loop** — `rewrite → tension → QG` (loop até SLO)
3. **War Room Pipeline** — `segment → offers → pulses → measure` (ciclo 24h)

1. **Caixa > Eloquência** — Toda saída avaliada por métrica financeira
2. **Prova > Promessa** — Evidência vem ANTES do pitch
3. **Kill > Insistência** — Não performa → kill automático
4. **Handoff > Herói** — Dor aguda sem fechamento = humano assume
5. **Compliance > Growth** — Segurança veta, crescimento espera
