# Squad Produto

> Roadmap, inovação e lançamento de produtos.

## Visão Geral

Squad responsável pelo ciclo completo de produto: discovery, especificação, criação, QA, lançamento e feedback loop. Recebe insights do CS e alimenta Marketing e Vendas com produtos prontos.

## Agentes

| Agente | Persona | Papel |
|--------|---------|-------|
| Nova | `produto-head` | AI Head — roadmap, priorização, alinhamento |
| Compass | `produto-pm` | Product Manager — coleta, brainstorm, specs |
| Canvas | `produto-creator` | Content Creator — conteúdo, vídeo, material |

## Workflow Principal

```
Discovery → Spec → QG1 → Criação → QG2 → QA → QG3 → Launch → Feedback Loop
```

## Cross-Squad

- **CS → Produto:** `feedback_collected` → backlog
- **Produto → Marketing:** `product_launched` → campanha de lançamento
- **Produto → Vendas:** `product_ready` → material de vendas atualizado
