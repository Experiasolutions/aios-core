# SEÇÃO 7 — A PIPELINE DE PRODUÇÃO PARA EXPERIA

## Workflow: Demo KAIROS (3 minutos)

```
1. ROTEIRO
   └─ KAIROS (Gemini) gera o script
   └─ Gabriel refina com voz natural

2. AVATAR/APRESENTAÇÃO
   └─ HeyGen: gravar 1 take com voice clone do Gabriel
   └─ OU: Gabriel grava real + CapCut edita com AI

3. CAPTURAS DE TELA
   └─ OBS Studio: gravar tela do Telegram/KAIROS ao vivo
   └─ CapCut: cortar + adicionar legendas + efeitos

4. TRILHA SONORA
   └─ Soundverse: background instrumental
   └─ Suno: jingle de abertura (5s)

5. MONTAGEM FINAL
   └─ CapCut: montar tudo + render final
   └─ Sem watermark, qualidade profissional
```

## Workflow: Reels Instagram (30-60s)

```
1. CONTEÚDO
   └─ KAIROS gera copy + hook
   └─ Gabriel grava webcam/celular (voz amantegada = DIFERENCIAL)

2. OPÇÃO A — Gabriel aparece (RECOMENDADO para confiança)
   └─ Gravar 2-3 takes com celular
   └─ CapCut: auto-edit, legendas, cortes, B-roll
   └─ Resultado: autêntico, humano, confiante

3. OPÇÃO B — AI puro (para volume de conteúdo)
   └─ Qwen AI / Kling: gerar clips visuais
   └─ ElevenLabs: narração com voz clonada
   └─ CapCut: montar + legendas

4. OPÇÃO C — Mix (ideal)
   └─ Gabriel abre com hook na câmera (5s)
   └─ IA gera o visual do restante (25s)
   └─ Narração ElevenLabs + trilha Soundverse
```

## Workflow: Quiz Gamificado (Lead Magnet)

```
1. LANDING PAGE
   └─ HTML custom no GitHub Pages

2. QUIZ
   └─ 8 perguntas sobre maturidade digital
   └─ Score + mini-relatório gerado pelo KAIROS

3. RESULTADO
   └─ Vídeo personalizado gerado por HeyGen/D-ID
   └─ OU texto + gráfico gerado pelo KAIROS

4. CAPTURA
   └─ Nome + WhatsApp + email
   └─ Lead cai no Google Sheets (CRM MED)
```
