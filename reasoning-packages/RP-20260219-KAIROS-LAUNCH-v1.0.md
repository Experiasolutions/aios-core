## BLOCO 3 — COLD CALL SCRIPT (45 min)

### Contexto do problema

Cold call para IA tem um desafio único: o interlocutor não sabe o que não sabe. Ele não está procurando o que você vende porque não imagina que existe.

O script não pode começar com "tenho uma solução de IA". Deve começar com a dor que ele já sente.

### Prompt para o KAIROS criar o script

```
@aios-master

Preciso de um script de cold call para a agência Kairos.

CONTEXTO:
  Quem eu sou: Gabriel, fundador da Kairos, especialista em
  implementação de IA para comércio no Grande ABC.
  
  Para quem ligo: donos de negócios locais no ABC
  (comércio, serviços, clínicas, restaurantes, etc.)
  
  O que ofereço: automação inteligente que substitui/potencializa
  a operação — atendimento, agendamento, marketing, financeiro —
  usando IA de nível enterprise, acessível para PMEs.

ESTRUTURA QUE PRECISO:

VERSÃO A — Cold call (nunca me viram):
  Duração alvo: 90 segundos até conseguir a reunião
  
  Objetivo: não vender. Conseguir 30 minutos de atenção.
  
  Estrutura:
  → Abertura que não parece vendedor (5 segundos)
  → Gancho com a dor específica do negócio (15 segundos)
  → Prova de que existe solução (10 segundos)
  → Pedido de reunião — simples e direto (10 segundos)
  → Resposta para as 3 objeções mais comuns (30 segundos cada)

VERSÃO B — Abordagem presencial (entro no estabelecimento):
  Para quando apareço pessoalmente
  Mais curto, mais direto, mais humano
  Objetivo: sair com o número ou com a data da reunião

VERSÃO C — WhatsApp (follow-up após contato inicial):
  Mensagem que não parece spam
  Que gera curiosidade real
  Que leva à reunião

AS 3 OBJEÇÕES PRINCIPAIS (e como responder):
  1. "Já tenho sistema / não preciso disso"
  2. "É muito caro / não tenho orçamento agora"
  3. "IA não funciona para meu negócio / é complicado"

Para cada objeção: a resposta que não defende, que concorda
parcialmente, e que redireciona para a dor real.

IMPORTANTE: o script deve soar como Gabriel — alguém que
acredita genuinamente no que faz, não como um vendedor
lendo roteiro. Deve ter espaços para improviso.
Deve assumir que o interlocutor é inteligente.

Convoque:
- Clone de Hormozi (oferta e fechamento)
- Clone de Chris Voss "Never Split The Difference" (negociação)
- War Room de Revenue
```
