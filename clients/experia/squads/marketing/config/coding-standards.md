# Coding Standards — Squad Marketing
## Naming
- Arquivos: `kebab-case.md`
- Agentes: `mkt-{role}.md`
- Tasks: `{verbo}-{objeto}.md`
## Task Format
- TASK-FORMAT-SPECIFICATION-V1 obrigatório
- Frontmatter YAML com: task, responsavel, Entrada, Saida, Checklist
## Métricas Obrigatórias
- Toda campanha deve ter KPIs definidos antes do launch
- Reports semanais com CTR, CPC, CPL, ROAS
