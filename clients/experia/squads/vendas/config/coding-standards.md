# Coding Standards — Squad Vendas
## Naming
- Arquivos: `kebab-case.md`
- Agentes: `vendas-{role}.md`
- Tasks: `{verbo}-{objeto}.md`
## Task Format
- TASK-FORMAT-SPECIFICATION-V1 obrigatório
## Pipeline Rules
- Todo lead deve ser scored antes de qualificar
- BANT obrigatório: Budget, Authority, Need, Timeline
- Proposal só após discovery call aprovada
