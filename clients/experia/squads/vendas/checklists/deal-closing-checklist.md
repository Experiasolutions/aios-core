# Checklist: Deal Closing

## Pré-Proposta
- [ ] Discovery call realizada e documentada
- [ ] Necessidades do cliente mapeadas (BANT completo)
- [ ] Decisor identificado e engajado
- [ ] Orçamento do cliente confirmado
- [ ] Timeline definida

## Proposta
- [ ] Proposta criada usando `proposal-tmpl.md`
- [ ] Valores alinhados com tabela de preços
- [ ] Escopo claramente definido
- [ ] Prazo de validade incluído
- [ ] Revisada pelo Head

## Negociação
- [ ] Objeções mapeadas e respondidas
- [ ] Desconto (se houver) dentro da política
- [ ] Aprovação interna obtida

## Fechamento
- [ ] Contrato revisado pelo jurídico (se aplicável)
- [ ] Assinatura obtida
- [ ] Pagamento confirmado/agendado
- [ ] Handoff para CS documentado (`deal_closed` event)
- [ ] CRM atualizado com status "Won"
