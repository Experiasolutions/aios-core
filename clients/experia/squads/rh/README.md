# 👑 AIOS Orion — Departamento de RH

Assistente inteligente de Recursos Humanos com IA.

## Como usar (3 passos)

### 1. Instale o Node.js
- Baixe em: [nodejs.org](https://nodejs.org) (versão LTS)
- Instale com "Next, Next, Finish"

### 2. Configure
- Abra o arquivo `.env` e preencha as chaves (peça ao Gabriel):
  - `CLICKUP_API_KEY` — Token do ClickUp
  - `GROQ_API_KEY` — Chave da IA

### 3. Inicie
- Dê dois cliques no arquivo **INICIAR.bat**
- Pronto! O assistente vai abrir no terminal.

## Comandos

| Comando | O que faz |
|---------|-----------|
| `funcionarios` | Lista todos os funcionários |
| `vagas` | Lista vagas em aberto |
| `criar vaga [cargo]` | Cria nova vaga |
| `admitir [nome]` | Registra novo funcionário |
| `ajuda` | Mostra todos os comandos |
| `sair` | Encerra o programa |

## Chat com IA

Além dos comandos, você pode **conversar naturalmente**:
- "Como calcular o turnover da empresa?"
- "Preciso de um modelo de advertência"
- "Quais são os direitos do trabalhador em férias?"
- "Monte um checklist de onboarding"

A IA responde com base na CLT e boas práticas de RH!
