# Checklist: Campaign QA

## Pré-Launch
- [ ] Público-alvo definido (persona + segmento)
- [ ] Budget aprovado pelo Head
- [ ] KPIs definidos (CTR, CPC, CPL, ROAS targets)
- [ ] Copy aprovada (A/B variations)
- [ ] Criativos revisados (formato, resolução, texto)
- [ ] UTMs configuradas corretamente
- [ ] Landing page testada (mobile + desktop)

## Launch
- [ ] Pixels de tracking ativos (Meta, Google)
- [ ] Campanha configurada conforme brief
- [ ] Bid strategy definida
- [ ] Agendamento correto
- [ ] Email de nurturing conectado ao fluxo

## Pós-Launch (D+3)
- [ ] Métricas iniciais coletadas
- [ ] Anomalias verificadas
- [ ] Ajustes de bid realizados (se necessário)
- [ ] Report resumido enviado ao Head
