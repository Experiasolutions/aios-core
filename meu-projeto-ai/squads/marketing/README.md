# Squad Marketing

> Demanda, tráfego, conteúdo e conversão.

## Visão Geral

Squad responsável por gerar demanda qualificada (MQLs) para o Squad Vendas. Opera todo o funil de marketing: pesquisa de mercado, campanhas pagas, email marketing, conteúdo e social media.

## Agentes

| Agente | Persona | Papel |
|--------|---------|-------|
| Pulse | `mkt-head` | AI Head — campanhas, budget, métricas |
| Ignite | `mkt-traffic` | Media Buyer — Meta Ads, Google Ads |
| Drip | `mkt-email` | Email — segmentação, automações |
| Scribe | `mkt-content` | Content — SEO, blog, reformatação |
| Prism | `mkt-social` | Social — calendário, posts, agendamento |

## Workflow Principal
```
Research → Brief → Campanha → Launch → Report → MQL Handoff
```

## Cross-Squad
- **Marketing → Vendas:** `lead_qualified` (MQL) → score-lead
- **Produto → Marketing:** `product_launched` → create-launch-campaign
- **Vendas → Marketing:** `nurture_request` → retorno ao nurturing
