# Offer Launch Checklist

> **Responsável:** @doom-master
> **Quando usar:** Antes de lançar qualquer oferta, campanha ou War Room.

## NMI Gates
- [ ] 7 etapas completas (Nicho → Dor → Gancho → Mecanismo → Prova → Oferta → CTA)
- [ ] Price Lab Express executado (preço validado por simulação)
- [ ] Micro-garantia condicional definida e caixa isolado

## Arena Gates
- [ ] Mínimo 3 variantes geradas para Arena de Ofertas
- [ ] Tournament executado (vencedora identificada)
- [ ] Kill rule definida (N derrotas = kill)

## Quality Gates
- [ ] Quality Gate score ≥ 8 em todas as rubricas
- [ ] Juiz de Caixa aprovado: P(aceite) × margem × payback > threshold
- [ ] Copy Review Loop completo (máx. 3 iterações)

## Revenue Gates
- [ ] Auto-P&L projetado com margem ≥ threshold
- [ ] Profit Firewall: brownout threshold definido
- [ ] CAC/LTV/Payback projetados e documentados

## Security & Compliance
- [ ] OPA gates verificados
- [ ] Privacy Budget checado (cota disponível)
- [ ] C2PA manifests gerados para todos os criativos
- [ ] Compliance check: sem promessas proibidas
- [ ] LGPD/GDPR: dados no escopo mínimo necessário

## Launch Prep
- [ ] Cash Pulse janelas definidas
- [ ] Métricas de tracking configuradas (TTF$, Taxa Aceite, Margem)
- [ ] Runbook de emergência pronto (se margem < threshold → brownout)
- [ ] Handoff humano configurado (se dor aguda + sem fechamento)

## GO / NO-GO
- **Todos os gates ✅:** → **GO** 🚀
- **Qualquer gate ❌:** → **NO-GO** 🚫 (resolver antes de launch)
