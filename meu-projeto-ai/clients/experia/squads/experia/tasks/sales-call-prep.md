# Prep de Call de Vendas

## Objetivo
Preparar todo material necessário para uma call de demonstração/venda do Sprint 14D.

## Inputs
- [ ] Nome da clínica
- [ ] Perfil Instagram (se houver)
- [ ] Especialidade (estética, odonto, dermato, etc.)
- [ ] Quem é o decisor (nome e cargo)
- [ ] Como chegou (prospecção ativa, indicação, inbound)

## Entregáveis

### 1. Briefing de Pesquisa (5 min)
- [ ] Checar Instagram: quantidade de seguidores, frequência de posts, qualidade
- [ ] Checar Google Business: nota, quantidade de reviews, reclamações comuns
- [ ] Checar WhatsApp: tempo de resposta (enviar mensagem teste)
- [ ] Identificar 3 pontos de dor específicos da clínica

### 2. Script Personalizado
- [ ] Adaptar o Script Religioso com dados reais da clínica
- [ ] Preparar 3 objeções prováveis e respostas
- [ ] Preparar pitch do bônus (Kit Instagram ou Kit Escritório)
- [ ] Ter downsell 7D pronto se precisar

### 3. Material de Apoio
- [ ] Link do Instagram da Experia (prova social)
- [ ] Exemplo de relatório semanal (template preenchido)
- [ ] Calculadora de ROI simplificada

## Pós-Call
- [ ] Registrar resultado na planilha (fechou / follow-up / perdido)
- [ ] Se fechou: iniciar `onboard-clinic.md`
- [ ] Se follow-up: agendar D+1 e D+3
- [ ] Se perdido: anotar motivo para melhoria contínua
