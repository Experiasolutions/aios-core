## Filosofias Core
- **Task-First:** Tasks vêm primeiro, agentes são executores
- **Quality Gates:** Score >70% para avançar
- **OPS = Arquiteto:** Define processos, outros squads executam
- **Decision Tree:** Roteamento automático de tasks entre squads
