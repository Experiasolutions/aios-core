## WAAS — Workforce as a Service

### Modelo de Negócio

```
Cliente contrata → Gabriel configura squad → AIOS executa 24/7
                                          → Gabriel monitora via Telegram
                                          → Cliente recebe dashboards
```

| Tier | Target | Squads | Preço/mês | Margem |
