# Tech Stack (Experia)

## Core
- **AIOS Framework:** v2.1.0+
- **Language:** Markdown (definições), JavaScript/Python (scripts)

## Integrações
- **Messaging:** WhatsApp Cloud API (via BSP ou Direto)
- **CRM:** Pipedrive, Kommo, ou Planilha (Google Sheets)
- **Automação:** Make.com (low-code), n8n (self-hosted)

## Dados
- **Formato:** JSON para payloads
- **Armazenamento:** Local (JSON/SQLite) ou Cloud (Postgres/Supabase)

## Segurança
- **Auth:** OAuth2 ou API Key (variáveis de ambiente)
- **Criptografia:** TLS 1.2+ em trânsito
