# Checklist: Onboarding de Cliente

## Pré-Kickoff
- [ ] Briefing de vendas recebido e revisado
- [ ] Acesso ao sistema criado
- [ ] Kick-off agendado com contato principal
- [ ] Material de boas-vindas preparado

## Kick-off (Dia 1)
- [ ] Apresentação da equipe/responsável
- [ ] Alinhamento de expectativas e metas
- [ ] Cronograma de primeiros 7 dias definido
- [ ] Canais de comunicação definidos

## Setup (Dias 2-3)
- [ ] Integrações ativas e testadas
- [ ] Dados iniciais importados
- [ ] Configurações personalizadas aplicadas

## First Value (Dias 4-7)
- [ ] Primeiro resultado entregue
- [ ] Cliente confirmou valor recebido
- [ ] Health score inicial registrado

## Handoff
- [ ] Cliente marcado como "Ativo" no sistema
- [ ] Responsável CS definido para acompanhamento
- [ ] Primeira reunião de acompanhamento agendada
