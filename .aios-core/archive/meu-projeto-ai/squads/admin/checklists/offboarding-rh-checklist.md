# Checklist: Offboarding RH

## Comunicação
- [ ] Comunicação formal ao colaborador
- [ ] Comunicação ao time
- [ ] Data de último dia definida

## Acessos
- [ ] Email corporativo desativado
- [ ] Acessos a sistemas revogados
- [ ] VPN removida
- [ ] Senhas alteradas em contas compartilhadas

## Patrimônio
- [ ] Notebook/equipamentos devolvidos
- [ ] Badge/crachá devolvido
- [ ] Chaves devolvidas

## Trabalhista
- [ ] Rescisão calculada por `@admin-dp`
- [ ] Termos revisados por `@admin-juridico`
- [ ] TRCT assinado
- [ ] Guias de saque FGTS emitidas
- [ ] Exame demissional realizado

## Conhecimento
- [ ] Entrevista de saída realizada
- [ ] Responsabilidades transferidas
- [ ] Documentação de projetos atualizada
- [ ] Handoff para substituto/time
