# Coding Standards — Squad OPS

## Naming
- Arquivos: `kebab-case.md`
- Agentes: `ops-{role}.md`
- Tasks: `{verbo}-{objeto}.md`

## Task Format
- TASK-FORMAT-SPECIFICATION-V1 obrigatório
- Frontmatter YAML com: task, responsavel, Entrada, Saida, Checklist

## Quality Gate
- Mínimo: 70% score
- Checklist: `quality-gate-70.md`
- Todo deliverable passa por QG antes de deploy
