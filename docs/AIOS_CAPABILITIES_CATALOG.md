## ⚡ 4. MOTORES TÉCNICOS (The Engine)

*   **RAG Engine (`rag-engine.js`):** "Google Interno". Busca informações em todos os seus arquivos (PDFs, Markdown, Código) em milissegundos.
*   **MCP Server (`mcp-server.js`):** "Conector Universal". Permite que o Claude Desktop ou Cursor usem seus agentes como ferramentas.
*   **Kernel Bridge (`kernel-bridge.js`):** A ponte que conecta o chat ao cérebro do sistema.
