## 🚀 1. DEPLOY NA VERCEL (Site e Quiz no Ar)

Você já tem os arquivos prontos, só precisa subir. Como você está no 3G (tethering USB), o deploy pelo terminal (`npx vercel`) é a forma mais leve e rápida (não consome quase nada de banda).

### Passo a Passo:
1. Abra o terminal do KAIROS.
2. Faça login na Vercel (se já não estiver logado):
   ```bash
   npx vercel login
   ```
   *(Ele vai abrir o navegador pro GitHub ou pedir seu email. É um processo bem leve).*

3. Faça o deploy da Landing Page:
   ```bash
   cd "c:\Users\Gabriel\Documents\My KAIROS\clients\experia\frontend"
   npx -y vercel --prod
   ```
   *(Aperte Enter para aceitar as opções padrão: scope, link to existing project? N. O link gerado será algo como `frontend-experia.vercel.app`).*

4. Faça o deploy do Quiz:
   ```bash
   cd "c:\Users\Gabriel\Documents\My KAIROS\clients\experia\quiz"
   npx -y vercel --prod
   ```
   *(O link gerado será algo como `quiz-experia.vercel.app`).*

**Pronto.** Salve esses dois links gerados.
